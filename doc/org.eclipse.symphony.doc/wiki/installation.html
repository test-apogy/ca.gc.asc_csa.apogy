General

3D Acceleration

Symphony makes use of 3D. It is important to install latest & stable version of graphics card 3D drivers.

Known Bugs

  1. export SWT_GTK3=0

Symphony RCP Installation & Configuration

All Symphony RCP applications are based on the same foundation and are packaged the same way. Symphony RCP applications are self-contained and are packaged in zip format. Symphony supports the following platform:

  • 32-bit Windows
  • 64-bit Windows
  • 32-bit Linux (GTK)
  • 64-bit Linux (GTK)

Java Virtual Machine

  1. Ensure a proper Java Runtime Environment (JRE) is installed on your system (1.8 or higher). You can download JRE from http://www.oracle.com/technetwork/java/javase/downloads/index.html.
  2. The virtual machine must be accessible in the PATH. Type the following commands in a system shell to verify the installed java version.
    java -version
openjdk version "1.8.0_51"
OpenJDK Runtime Environment (build 1.8.0_51-b16)
OpenJDK 64-Bit Server VM (build 25.51-b03, mixed mode)

Symphony Rich Client Platform Applications

  1. Install Symphony RCP Application
    unzip symphony.product-linux.gtk.x86_64.zip
  2. Launch Symphony RCP Application
    cd symphony
./symphony (Symphony executable)

Select Workspace

  1. The Workspace Launcher dialog will prompt. Select the location of the new or the existing workspace. The workspace will contain the information related to your Symphony session.

Symphony SDK Installation & Configuration

Eclipse

Symphony is based on Eclipse. The first installation step is the proper installation and configuration of Eclipse. Eclipse configuration details can be found on https://www.eclipse.org.

Oomph

To install Eclipse platform, we highly recommend to use Eclipse Installer also known as Oomph.

  1. Get the Eclipse Installer from https://wiki.eclipse.org/Eclipse_Oomph_Installer
  2. Extract the archive under <ECLIPSE_INSTALLER_HOME>.
  3. Ensure a proper Java Runtime Environment (JRE) is installed on your system (1.8 or higher). You can download JRE from http://www.oracle.com/technetwork/java/javase/downloads/index.html
  4. Copy <JRE_HOME> directory into <ECLIPSE_INSTALLER_HOME> directory. You may also use a symbolic link. 
    eclipse
  configuration
  eclipse
  eclipse.ini
  features
  plugins
  jre      <<<< must include /bin/java
  ...
  5. Launch <ECLIPSE_INSTALLER_HOME>/eclipse
  6. Select Eclipse Modeling Tools
  7. Select Latest Release (Mars) as Product Version
  8. Select a proper JRE 1.8+
  9. Set the Installation Folder (e.g. <ECLIPSE_HOME>)
  10. Click on Install
  11. Once completed, edit <ECLIPSE_HOME>/eclipse.ini and change the following lines that set VM memory parameters (see below). 
    -Xms256m                              <<<< Replace by -Xms2048m 
-Xmx1024m                             <<<< Replace by -Xmx2048m
  12. Save your changes.
  13. Launch the <ECLIPSE_HOME>/eclipse.
  14. The Workspace Launcher dialog will prompt. Select the location of a new or an existing workspace. The workspace will contain the information related to your Symphony session.

Install Eclipse Features

Eclipse Mars Packages

  1. Install the following packages:
    • Modeling \ EMF - Eclipse Modeling Framework Xcore SDK

Sirius Packages

  1. Add the Sirius P2 Site http://download.eclipse.org/sirius/updates/releases/3.1.1/mars.
  2. Install the following packages:
    • Sirius/Sirius Core Runtime
    • Sirius/Sirius Runtime Support for Acceleo 3

Orbit Packages

  1. Add the Orbit P2 Site http://download.eclipse.org/tools/orbit/downloads/drops/R20150821153341/repository/
  2. Install the following packages:
    • Orbit Bundles By Name: org.apache.*/org.apache.commons.collections (Be careful select the one that is not the source plugin (1st on top))
    • Orbit Bundles By Name: org.apache.*/org.apache.commons.net
    • Orbit Bundles By Name: org.apache.*/org.apache.commons.pool

Nebula Packages

  1. Add the Nebula P2 Site http://download.eclipse.org/technology/nebula/snapshot
  2. Install the following packages:
    • Nebula Release Individual Widgets/Nebula Gallery Widget

Install Symphony

Symphony is composed of a series of Eclipse features. The following steps present the way to install them.

Third-Party Packages

  1. Clone Symphony Third-Party repository 
    git clone 
				https://<user>@bitbucket.org/Symphony2Eclipse/org.eclipse.symphony.ots.git
  2. Create a directory to store a backup of pre-compiled Symphony Third-Party Packages.
    cd ~; mkdir tmp
  3. Make a temporary copy of symphony_ots_repository.
    cp git/org.eclipse.symphony.ots/symphony_ots_repository_0_1.tar.gz ~/tmp
  4. Extract the content 
    cd ~/tmp; 
tar -xvf symphony_ots_repository_0_1.tar.gz
  5. Launch the Eclipse executable.
    cd <ECLIPSE_HOME>
./eclipse
  6. Select the workspace location.
  7. Add the Symphony Third-Party Local P2 Site.
  8. Select and install all the packages.

Symphony Packages

  1. Clone Symphony repository 
    git clone 
				https://<user>@bitbucket.org/Symphony2Eclipse/org.eclipse.symphony.git
